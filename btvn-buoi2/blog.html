Hướng dẫn xây dựng ứng dụng Todo List bằng HTML, CSS & JavaScript

Todo List là một ứng dụng rất phổ biến cho người mới học lập trình web vì nó giúp bạn:

  • Hiểu cấu trúc HTML
  • Làm quen CSS để tạo giao diện
  • Thực hành JavaScript với DOM, sự kiện và xử lý dữ liệu

Mục tiêu của bài này:

  • Nhập công việc mới
  • Hiển thị danh sách công việc
  • Xóa từng công việc khi không cần nữa

Trước tiên, ta tạo cấu trúc project đơn giản:

Todo App

  1. index.html
  2. style.css
  3. script.js

Xây dựng giao diện HTML

HTML dùng để tạo:

  • Ô nhập công việc
  • Nút thêm
  • Danh sách hiển thị các công việc

Code HTML (index.html)

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Todo List</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="todo-container">
        <h1>Todo List</h1>
        <div class="todo-input">
            <input type="text" id="todoInput" placeholder="Nhập công việc...">
            <button id="addBtn">Thêm</button>
        </div>
        <ul id="todoList"></ul>
    </div>
    <script src="script.js"></script>
</body>
</html>

Giải thích

  • <input> : nơi người dùng nhập công việc
  • <button> : nút để thêm công việc mới
  • <ul> : danh sách hiển thị các công việc

Trang trí giao diện bằng CSS

Mô tả: CSS giúp giao diện:

  • Đẹp mắt hơn
  • Phù hợp với thiết bị di động
  • Thiết kế dễ sử dụng

Code CSS (style.css)

 body {
    font-family: Arial, sans-serif;
    background: #f4f4f4;
}

.todo-container {
    width: 350px;
    margin: 100px auto;

    border-radius: 6px;
}

.todo-container h1 {
    text-align: center;
}

.todo-input {
    display: flex;
    gap: 10px;
}

.todo-input input {
    flex: 1;
    padding: 8px;
}

.todo-input button {
    padding: 8px 12px;
    cursor: pointer;
}

ul {
    list-style: none;
    padding: 0;
    margin-top: 20px;
}
li {
    display: flex;
    justify-content: space-between;
    padding: 8px;
    border-bottom: 1px solid #ddd;
}
li button {
    background: red;
    color: #fff;

    cursor: pointer;
}

Giải thích

  • .todo-container: căn giữa ứng dụng
  • display: flex: giúp <input> và <button> nằm cùng hàng <li>: mỗi công việc là một dòng riêng
  • <li> mỗi công việc là một dòng riêng

Xử lý logic bằng JavaScript

Mô tả: JavaScript sẽ:

  • Lấy dữ liệu từ <input>
  • Tạo phần tử <li> mới
  • Gắn sự kiện xóa cho từng công việc

Code JavaScript (script.js)

const input = document.getElementById("todoInput");
const addBtn = document.getElementById("addBtn");
const todoList = document.getElementById("todoList");

addBtn.addEventListener("click", function () {
    const text = input.value.trim();

    if (text === "") {
        alert("Vui lòng nhập công việc!");
        return;
    }

    const li = document.createElement("li");
    li.innerHTML = `
         <span>${text}</span>
        <button>Xóa</button>
    `;

    li.querySelector("button").addEventListener("click", function () {
        li.remove();
    });

    todoList.appendChild(li);
    input.value = "";
});

Kết quả đạt được

Sau khi hoàn thành:

  • Người dùng nhập công việc
  • Nhấn Thêm → công việc xuất hiện
  • Nhấn Xóa → công việc biến mất
